\# Cyber Threat Detection using XGBoost



This project builds a Machine Learning model using XGBoost on the NSL-KDD dataset to classify network traffic as Normal or Attack.



\## Features

\- Data preprocessing

\- One-hot encoding

\- XGBoost model training

\- Confusion matrix visualization

\- ROC curve plotting

\- Model saving using joblib



\## Technologies Used

\- Python

\- Pandas

\- Scikit-learn

\- XGBoost

\- Matplotlib



\## Dataset

NSL-KDD Dataset



\## How to Run



```bash

pip install -r requirements.txt

python train\_model.py
